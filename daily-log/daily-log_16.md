    실제 경기에 직접 뛰는 것이 아니라, 사용자가 실제 NFL선수들을 골라 가상의 팀을 만들고,

- yield
  : 값을 반환하고 함수를 그 자리에 일시정지
    처음부터 실행하는게 아니라 멈췄던 그 다음줄 부터 이어서 실행

- session.flush()
  : SQLAlchemy 작업공간에 쌓여있는 데이터 변경 사항(추가,수정, 삭제)을 실제 데이터베이스 서버로 전송하여 미리 동기화 하는 명령어
- 
  ``` 
  session.add() -> 파이썬 메모리에만 데이터 올려둠 (DB는 아직 모름)
  session.flush() -> DB 서버로 데이터 전송. 임시 번호표(ID) 발급 완료 (취소 가능)
  session.commit() -> 데이터 최종 금고에 저장 및 잠금장치 완료 (취소 불가능)
  ```
