2) SQL 기초 학습
- JOIN에 관한 문제 풀기
    - 서로 다른 데이블들을 하나로 묶어 하나의 큰 테이블로 만드는 기술
    - JOIN을 하려면 두 테이블 사이에 반드시 공통된 연결고리 컬럼이 존재 
    - PK와 FK가 만나는 지점 (기본키와 외래키의 만남)
    - INNER JOIN : 양쪽 테이블에 모두 데이터가 일치하는 교집합
    - OUTER JOIN : 일치하는 데이터가 없는 한쪽 테이블의 데이터까지 모두 포함
        - 데이터가 없는 빈칸은 : NULL로 표현

    ```SELECT 조회할_컬럼
    FROM 테이블A
    [INNER / LEFT / RIGHT] JOIN 테이블B
    ON 테이블A.공통컬럼 = 테이블B.공통컬럼;
    ```
